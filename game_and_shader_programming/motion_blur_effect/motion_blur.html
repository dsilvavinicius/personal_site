In this project the Post-processing Motion Blur effect explained in GPU Gems 3's chapter 27 is implemented and incorporated into Unity 3D's Boot Camp scene. Basically the shader uses the Depth Buffer to unproject pixels' uv coordinates back to world coordinates and reproject it using the model-view-projection matrix from the previous frame. The results are two uv coordinates, one from the current frame and another from the previous frame. The subtraction of these uv coordinates is equal to the pixel velocity, which is used to find pixel samples that contribute to the final pixel color. These contributions result in the blur effect. The effect can be softened or hardened depending on the number of samples used and the velocity's norm.

The source code contains just the shader and the associated JavaScript scripts. To use the effect, just attach all scripts to a camera and ensure that the shader property of the MotionBlur script is set with the correct shader.
