This effect simulates an ink press and can be used as a transition between images. It is very simple and is used on Northway Games' Incredipede. They briefly explained the arts and effects on this GDC 2013 talk. The shader just computes the brightness for each pixel and ensure that only the ones with brightness less than a defined threshold are rendered with their original colors. The others are clean to white or to another texture. To achieve a more smooth transition, an auxiliary image which consists of the blurred original image is used to calculate the pixel brightness instead of the original one. The blurring can be achieved with a Gaussian filter. The animation is done by changing the threshold. In this implementation the threshold is smoothly changed from 0.0 to 1.0 using the cos() function.

The source code contains just the shader and the associated JavaScript scripts. To use the effect, just attach all scripts to a camera and ensure that the shader property of the InkPress script is set with the correct shader.
