In this project the Old Film effect shader explained in this awesome Tutorial is implemented and incorporated into Unity 3D's Boot Camp scene. To achieve the effect, first, the image is converted to grayscale and a sepia overlay is applied. Second, noise is generated using the 2D simplex noise algorithm implemented by Ian McEwan from Ashima Arts. Third, to create the scratch effect, a serie of sine bands are generated using the sine function. The bands are distorted by a turbulence factor, which is calculated using the noise function. Finally, the vignetting effect is used to simulate the light occlusion that the lens can generate on the borders of the image. Two radius from the center of the image are defined: outer and inner. If the pixel distance from the center is greater than outer, the black color is returned. If the distance lays between inner and outer, the color is darkened by an interpolated multiplier. If the distance is less than inner, then the original color is returned. For more details and formulas, see the Tutorial above.
